Photoplop - April 2015

Photoplop was built as my final project during Dev Bootcamp and was created by myself and two other students in less than eight days. Photoplop is a mobile-first web app built with Ruby On Rails, that enables users to collaboratively build photo albums. Other technologies used include Cloudinary, Instagram API, Foundation and Omniauth.

I designed the user interface and implemented the front end design using HTML, SCSS, the Foundation framework, and Javascript. I was also involved in other aspects of the application, such as integrating with the Instagram API, database construction, and back end coding in Ruby on Rails. I led the creation of user flow maps, created mockups, and scripted and led user testing.
