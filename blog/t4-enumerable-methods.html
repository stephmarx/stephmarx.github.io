Enumerable#map:
Our New Best Friend

Sunday, December 14, 2014

An Enumerable is a module in Ruby that allows you to work with a collections by searching, sorting, or manipulating them. Sure, we can do a lot of the same things with arrays or hashes, but enumerables allow us to do the same thing with far fewer lines of code in a much simpler way. As an example, I'll be illustrating the Enumerable#map method.

Let's imagine we have a bunch of data in an array that we need to convert to integers. How might we do this with a simple loop and an array?

scores = [30.875, 28.522222, 26.001, 32.543, 31.879]
for x in 0..(scores.length-1)
scores[x] = scores[x].to_i
end

This would return each value in the array converted to an integer:

[30, 28, 26, 32, 31]

Hm, not ideal. There's an easier way to do this, though!

scores.map {|value| value.to_i}

This would return the same array as our first example, and it does it in just one line of code! There are lots more enumerables available for us to use in Ruby, but they all allow us to do something to every element of our collection at once, without using some kind of loop. Neat, huh?
