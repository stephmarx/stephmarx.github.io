Numbers

upto(limit){|i| block}

Iterates the given block, passing increasing values from the integer up to and including the limit.

        1.upto(5) {|n| print n, " "}
        #=> 1 2 3 4 5

downto(limit){|i| block}

Iterates the given block, passing decreasing values from the integer down to and including the limit.

        10.downto(6) {|n| print n, " "}
        #=> 10 9 8 7 6

even?

Returns true if the integer is an even number

odd?

Returns true if the integer is an odd number

round([ndigits])

Rounds the integer to a given precision in decimal digits, with a default of 0 digits. Precision may be negative. Returns a floating point number when ndigits is positive, self for zero, and rounds down for negative.

      1.round       #=> 1
      1.round(2)    #=> 1.0
      15.round(-1)  #=> 20

times {|i| block}

Iterates the given block a number of times equal to the integer, passing in values from zero to int - 1.

        4.times do |n|
          print n, " "
        end
        #=> 0 1 2 3

fixnum <=> number

Compares the fixnum to another number. Returns -1, 0, +1, or nil depending on whether the fixnum is less than, equal to, or greater than numeric. nil is returned if the two values can't be compared.

abs

Returns the absolute value of a fixnum.

Strings

Replace

Replaces the existing content of a string with the contents you want to replace it with

      For example: 
          string = "This is going to be replaced!"
          string.replace = "I want Chicken fingers!"

dInsert

Interestingly enough strings act much like arrays, they too can be indexed. The insert method is an instance when string indeces are useful, remember that negative indices refer to the end of the string! In this method you first type out your orginal string and then you follow it with the what you wish to insert and the index at which you would like to insert it. 

 
          For example:
         "tortilla".insert(-1," soup")  # => tortilla soup 
         "totilla".insert(-2,"soup" # => tortillsoupa

Split

This is a pretty handy method, it takes a string and converts it into an array, by splitting the string. Instances where we have used this in the past include thhe comma challenge and teh credit card challenge

    For Example:       
       def seperate(integer)
       result = integer.to_s.reverse.split(//)
       end 
    
     print seperate(12345) # =>["1","2","3","4","5"]

Substitution

The gsub method can be used for substitution. This method returns a string with all the substitutions specified incorporated. For isntance if one wanted to replace all the vowels in a particular word like venom they could do that by:

      replace = "venom".gsub(/[aeiou]/,"*")

Arrays

Don't forget: you can use -1 to indicate the last element of an array! -2 refers to the second last element of the array and so on.

take

Returns the first n elements of an array

        numbers = [1, 2, 3, 4, 5, 6]
        numbers.take(4)
        #=> [1, 2, 3, 4]

drop

Returns the elements after n elements have been dropped

        numbers = [1, 2, 3, 4, 5, 6]
        numbers.drop(4)
        #=> [5, 6]

empty?

Checks whether an array has any elements

include?(item)

Checks whether a particular item is included in the array

shift

Removes and returns the first item of the array

unshift

Adds a new item to the beginning of the array.

        nums = [1, 2, 3, 4]
        nums.unshift(2)
        #=> [2, 1, 2, 3, 4]

insert

Adds a new element to the array at any position. Can also be used to insert multiple values at once at the same position.

        nums.insert(4, "a number") #=> [1, 2, 3, 4, "a number"]

delete_at

Deletes an element at a particular index

        nums = [1, 2, 3, 4]
        nums.delete_at(2) #=> 3

delete

Deletes a particular element anywhere it is located in the array

        array = [1, 2, 1, 3, 4, 1]
        array.delete(1)
        array #=> [2, 3, 4]

uniq

Returns array with only unique values remaining. Can be used destructively or non-destructively.

        array = [1, 2, 3, 3, 4, 4, 7, 7, 7]
        array.uniq!
        array #=> [1, 2, 3, 4, 7]

select / reject

Can be done destructively or non-destructively. Selects or rejects items that meet a criteria from a block of code.

        array = [1, 2, 3, 4, 5]
        array.select { |i| i > 2}  #=> [3, 4, 5]
        array.reject { |i| i > 2}  #=> [1, 2]

Set intersection with &

Returns a new array contianing elements common to the two arrays and excluding any duplicates.

        [1, 1, 3, 5, 6] & [1, 3, 3, 3] #=> [1, 3]

Repetition with *

With a string argument, this is equivalent to array.join(str). Otherwise, it returns a new array built by concatenating the int copies of self.

        [1, 3, 5] * 3     #=> [1, 3, 5, 1, 3, 5, 1, 3, 5]
        [1, 3, 5] * ", "   #=> "1, 3, 5"

at(index)

Returns the element at index. Negative index counts from the end of the array. Returns nil if the index is out of range.

        nums = [1, 2, 3, 4, 5]
        nums.at(0)  #=> 1

collect{|item| block}

Incokes the block once for each element of the array and creats a new array containing the values returned by the block.

        nums = [1, 2, 3, 4]
        nums.collect { |n| n*3 } #=> [3, 6, 9, 12]

combination(n){|c| block}

Returns all combinations of length n of elements from the array.

        nums = [1, 2, 3, 4]
        nums.combination(1).to_a  #=> [[1], [2], [3], [4]]
        nums.combination(4).to_a  #=> [[1, 2, 3, 4]]

count

Returns the number of elements. If an argument is given, counts the number of elements which equal the object.

        nums = [1, 2, 3, 3, 4]
        nums.count                  #=> 5
        nums.count(3)               #=> 2
        nums.count {|n| n % 2 == 0} #=> 2

Hashes

Hash syntax and Key value pairs

Remeber hashes can be represented in a number of ways syntactically! Different syntax can effect how the key value pairs are accessed. Don't be alarmed! 
         For example: 
         
        menu = {"Ramen" => 15, "Chicken Tikka" => 10, "Vegetable Samosa" => 2}
        menu = {:ramen  => 15, :chicken_tikka => 10, vegetable_samosa =>2 }
        menu = {ramen: 15, chicken_tikkah: 10, vegetable_samose:2}
        to get the value for a certain key: menu[:ramen] # => 15 
        to get the values of the hash and place them in an array: menu.values [15, 10 , 2]
        to get the name of all the keys and place them in an array  : menu.keys #=> ["ramen"/ "Ramen", "chicken_tikkah"/"Chicken Tikkah" , "vegetable_samosa"/"Vegetable Samosa"]
        to check weather or not a paricular value is present in this hash: menu.has_value(2) #=> true. If a number such as 100 was put in place of 2 the method would retun false, because 100 is not a valid value in the hash.
